## 🎯 NÄMÄ GURÝARSYŇ

Toý mekanynyň işgärleri we dolandyryjylary üçin **doly admin paneli**.  
Bu panel öňki döredilen `index.html` toý mekany saýtynyň ähli maglumatlaryny,  
bronlaryny, maliýe hasaplaryny we analitikasyny dolandyrýar.

**Çykyş:** Ýeke `admin.html` faýly — hiç hili backend ýok.  
Ähli maglumatlar `localStorage`-da saklanýar.  
Real işe goýlanda bu panel arka tarap (backend) bilen çalşylmaga taýýar arhitekturada ýazylmaly.

```javascript
function checkConflicts(bookings) {
  return bookings.filter((b1, i) =>
    bookings.some((b2, j) =>
      i !== j &&
      b1.hall === b2.hall &&
      b1.weddingDate === b2.weddingDate &&
      b1.status !== 'cancelled' &&
      b2.status !== 'cancelled'
    )
  );
}
```

## 💾 localStorage MAGLUMAT DOLANDYRMA

```javascript
// Merkezi maglumat dolandyryjy
const DB = {
  get: (key) => {
    try {
      return JSON.parse(localStorage.getItem('venue_' + key)) || [];
    } catch { return []; }
  },
  
  set: (key, data) => {
    localStorage.setItem('venue_' + key, JSON.stringify(data));
  },
  
  add: (key, item) => {
    const items = DB.get(key);
    items.push({ ...item, id: Date.now(), createdAt: new Date().toISOString() });
    DB.set(key, items);
    return items;
  },
  
  update: (key, id, updates) => {
    const items = DB.get(key).map(item =>
      item.id === id
        ? { ...item, ...updates, updatedAt: new Date().toISOString() }
        : item
    );
    DB.set(key, items);
    return items;
  },
  
  remove: (key, id) => {
    const items = DB.get(key).filter(item => item.id !== id);
    DB.set(key, items);
    return items;
  }
};

// Başlangyçda demo maglumatlary ýükle (ilkinji açylanda)

  }
}
```
